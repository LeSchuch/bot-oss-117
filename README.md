## Installation

```bash
git clone https://github.com/SchuchDev/bot-os-117
cd bot-os-117
```

Create a python virtual environment, and activate it:

```bash
virtualenv venv
source venv/bin/activate
```

Install python modules:

```bash
pip install -r requirements.txt
```

Run script:

```bash
python3 main.py
```
